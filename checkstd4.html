กรุณาเลือกห้องเรียนเพื่อแสดงรายชื่อนักเรียน

ตัวกรองข้อมูล
ไม่พบข้อมูลตามเงื่อนไขที่ค้นหา
ตั้งค่ารับการแจ้งเตือนผ่าน LINE Official
วิธีรับค่า User ID (UID) สำหรับการแจ้งเตือน
  1. เพิ่มเพื่อน LINE Official Account ของโรงเรียน (แสกน QR Code หรือเพิ่มจาก ID ที่โรงเรียนกำหนด)
  2. เข้าไปที่ห้องแชทของโรงเรียน และพิมพ์ข้อความว่า "ขอรหัส" หรือกดที่เมนูด้านล่าง
  3. ระบบจะตอบกลับพร้อมแจ้ง LINE User ID ของท่าน (รหัสจะขึ้นต้นด้วยตัวอักษร U ตามด้วยตัวเลขและภาษาอังกฤษ)
  4. คัดลอกรหัส User ID นั้นมาวางในช่องด้านล่างนี้
ระบบจะส่งแจ้งเตือนการเข้าแถวของบุตรหลานไปยัง LINE ของท่านอัตโนมัติ
